# AVO-ai

## 🥑 AVO
> #### ▶︎ 청각장애를 가진 부모님들의 육아를 도와주는 서비스  <br>
> #### ▶︎ 사용자 맞춤 진동 서비스. 각 상황별로 사용자가 설정한 진동과 함께 푸시알림을 보내주는 기능 탑재
>       - 우는 상황, 소리지르는 상황, 엄마,아빠를 부르는 상황: 총 3가지 상황
>       - 한 번 진동, 두 번 진동, 세 번 진동, 연속 진동: 총 4가지 진동 설정 가능
